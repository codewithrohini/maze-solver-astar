# 🧩 Maze Solver — A* vs BFS vs Dijkstra

Enhanced maze pathfinding project built with Python and pyamaze.

## ✨ Features
- Three algorithms running simultaneously — A*, BFS, Dijkstra
- Selectable heuristics — Manhattan, Euclidean, Chebyshev
- Weighted terrain — Road, Grass, Swamp with different move costs
- Color-coded paths — Red (A*), Blue (BFS), Green (Dijkstra)
- Stats dashboard — path length, nodes explored, time, efficiency score

## 🚀 How to Run
```bash
pip install pyamaze
python maze_astar_enhanced.py
```

## 🛠 Built With
- Python 3
- pyamaze
- tkinter
